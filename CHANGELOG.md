# Changelog

All notable changes to the Top 100 Million Domains dataset will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [November 2025] - 2025-11-15

### Added
- Initial release of Top 100 Million Domains dataset
- 6 subset files (1k, 10k, 100k, 1M, 10M, 100M)
- Comprehensive documentation:
  - README with quick start guide
  - Methodology explanation
  - Use cases and examples
  - FAQ
- Example code in Python, JavaScript, and PHP
- MIT License

### Data Stats
- **Total domains**: 100,000,000
- **Data sources**: Common Crawl (latest release) + Internal signals
- **Top domain**: google.com (DA: 996)
- **Median DA**: 56
- **TLD distribution**:
  - .com: 75.2%
  - .org: 8.1%
  - .net: 4.3%
  - Other: 12.4%

### Notes
- First public release
- Ranking methodology v1.0
- Based on Common Crawl data through October 2025
- RDAP data through November 2025
